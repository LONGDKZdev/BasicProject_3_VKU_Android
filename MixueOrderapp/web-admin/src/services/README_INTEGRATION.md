# Integration points (UI-only skeleton)

This `web-admin` folder is kept **UI-only**.

To integrate with Firebase/your backend later, replace the in-memory services in:
- `src/services/authService.js`
- `src/services/productsService.js`
- `src/services/ordersService.js`

Keep the function signatures the same so `src/main.js` (UI layer) does not need changes.
