Đăng nhập Admin

Đăng nhập bằng Firebase Auth. Quyền admin lấy từ users/{uid}.role = ADMIN.

Admin mới?

Mã mời dùng để xác thực quyền truy cập web-admin. Tài khoản vẫn đăng nhập bằng Firebase Auth.

Quyền hiện tại

UID
-
Email
-
Role
-

Healthcheck / Seed

Khu vực này để test nhanh (tuỳ chọn). Nếu cần seed/healthcheck thật, sẽ implement sau.
