## 🚀 CÁC BƯỚC TIẾP THEO

### 1. Build & Test
```bash
./gradlew build
./gradlew installDebug
```

### 2. Kết Nối Firebase Auth
```kotlin
// Trong LoginScreen.kt
val auth = FirebaseAuth.getInstance()
auth.signInWithEmailAndPassword(email, password)
```

### 3. Load dữ liệu Firestore
```kotlin
// Trong HomeViewModel.kt
val db = FirebaseFirestore.getInstance()
db.collection("products").get()
```

### 4. Xóa File Cũ (Safe)
- Một khi Firebase được kết nối, có thể xóa:
  - `ui/auth/` (auth screens)
  - `ui/cart/` (cart fragment)
  - `ui/home/` (home fragment)
  - `ui/order/` (order fragment)
  - Tất cả `res/layout/*.xml`
